# Mobile App Template

A clean, modern React Native template pre-configured with:
- **Expo Router** (File-based routing)
- **Uniwind** (Tailwind CSS styling)
- **i18next** (Internationalization)
- **Keycloak** (Authentication 

1.  **Enter Directory**:
    ```bash
    cd my-new-app
    ```

2.  **Configuration**:
    - Copy `.env.example` to `.env`.
    - Fill in your Keycloak details.

3.  **Run**:
    ```bash
    npm run dev
    ```

## Features

- **Authentication**: Built-in `AuthContext` with Keycloak integration.
- **Language**: Persistent language selection (System/EN/HE).
- **Theme**: Dark/Light mode support with persistent toggle.
- **Profile**: Ready-to-use profile tab with settings.
